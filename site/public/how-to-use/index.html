devAny - How to use devAny

devany command

devany command is available and this command opens file or link to Editor and Browser respectively.

Open file with Editor from Terminal

 $ devany -f content/how-to-use/index.md

Open link with Browser from Terminal

 $ devany -l https://www.google.com

Shortcuts

I know programmer wants to use keyboard as often as possible. Therefore, devAny provides many shortcuts. You can see them on the top bar.

You can modify these shortcusts in configuration

Common

Move Window : Alt+Tab

Full Windows : Alt+Shift+?

Editor

Copy : Alt+c

Paste Alt+v

Save : Alt+s

New file : Alt+n

Close file : Alt+q
